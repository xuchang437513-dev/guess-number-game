游戏规则

快速了解 1A2B 猜数字

🎯 游戏目标

计算机随机生成一个 3-6位不重复 的数字（可自定义位数），你的任务是 猜出这个数字 是什么。

🔢 如何猜测

输入你猜测的数字，每位必须是 0-9 且 不能重复。

1234 ✓ 有效
5678 ✓ 有效
1122 ✗ 有重复
12345 ✗ 位数不对

📊 结果含义

每次猜测后，系统会返回 XA YB 格式的结果：

🎯 A = 位置正确的数字
🔶 B = 数字正确但位置错误

⚙️ 难度设置

游戏提供 3种难度 供选择：

🌱 入门 无限次数，轻松游玩
⚡ 普通 限制20次机会
🔥 困难 仅15次机会，刺激加倍

👥 多人对战

在同一台电脑上与好友对战！2-4名玩家轮流猜测同一个神秘数字，先猜出者获胜，机会用完则淘汰！

👥 玩家数量 2-4人
🟢 简单 每人20次机会
⚡ 普通 每人15次机会
🔥 困难 每人15次机会
💀 地狱 每人5次机会

🏆 计分规则

每局游戏结束后，根据表现获得积分！

🎯 核心公式 基础分 × 效率系数 × 位数系数 + 奖励
💯 基础分 100 分
⚡ 效率系数 (N-k+1)/N（次数越少系数越高）
🔢 位数系数 3位1.0x / 4位1.3x / 5位1.6x / 6位2.0x
🚀 快速命中 1次+100 / 2次+60 / ≤3次+40
🎯 精准奖励 每次A×3分（每次接近答案）
📈 进步奖励 每次比上次更接近+5分
⭐ 连胜奖励 2连+20 / 3连+40 / 5连+100
❌ 失败 得0分（不扣分）

🎓 猜数字技巧

• 先用不重复的数字试探，观察反馈

• 注意 1A0B 说明数字对但位置全错

• 利用排除法缩小范围

• 最佳成绩通常在 位数~位数+4 次之间
