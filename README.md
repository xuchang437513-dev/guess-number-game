# 1A2B 猜数字游戏

一个精美的 1A2B 猜数字对战游戏，支持单人和多人模式。

## 游戏规则

1. 系统随机生成一个不重复的数字
2. 玩家输入猜测的数字
3. 系统返回 XAYB 格式的结果：
   - **A** = 位置正确且数字正确的个数
   - **B** = 数字正确但位置错误的个数
4. 根据结果继续猜测，直到猜出正确答案

## 功能特点

- 多种难度选择（入门、普通、困难、自定义次数）
- 支持 3-6 位数字
- 多人对战模式（2-4人）
- 积分系统
- 历史排行榜
- 响应式设计，支持移动端

## 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏。
